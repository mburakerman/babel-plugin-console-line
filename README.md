# `babel-plugin-console-line`

Your `console.log` ⏬

```
console.log("hello")
```

Now now with line info ✨

```
console.log("(line:1)", "hello")
```

### ⬇️ Install

```
npm i -D babel-plugin-console-line
```

### 🪡 Usage

```
{
   "plugins": ["babel-plugin-console-line"]
}
```

### ✅ Test

Clone project and run `npm run test`
